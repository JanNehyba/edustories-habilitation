# Skripty (veřejná podmnožina)

Obsahuje kontrolní brány (95, 96), generátory manifestů/přílohy C, figury
a definici řezu K1. Skripty přípravy surových dat nejsou veřejné: vyžadují
neveřejné datové řezy (nezveřejněné kazuistiky) a obsahují interní mapování
pseudonymů anotátorek; jejich logiku dokumentuje kniha (kap. 4, příloha C).
