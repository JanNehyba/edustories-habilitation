# Veřejná datová podmnožina

Odvozené tabulky bez textů kazuistik a bez osobních údajů (anotátorky pod
pseudonymy A1–A12). Plné datové řezy obsahují i nezveřejněné kazuistiky,
a proto součástí companionu nejsou; veřejný korpus je na Hugging Face
(MU-NLPC/Edustories-en). Významy sloupců dokumentuje kniha (kap. 4, příloha C).
