# Edustories: companion repozitář habilitační práce

Doprovodný repozitář monografie *Náročné chování žáků v době AI: kazuistiky
z praxe a jazykové modely v pedagogickém výzkumu*
(Jan Nehyba, Masarykova univerzita, 2026).

Obsah: analytické skripty a notebooky (EN), manifesty čísel (jediný zdroj
čísel prózy), HTML reporty analýz, figury, veřejná datová podmnožina
(pseudonymy A1–A12, bez textů kazuistik) a build knihy. Reprodukce:
`./reproduce.sh`. Veřejný korpus: Hugging Face `MU-NLPC/Edustories-en`.

**Jak si ověřit čísla knihy** (i bez programování): viz
[JAK-OVERIT-CISLA.md](JAK-OVERIT-CISLA.md). Analýzu Studie 3 lze spustit
přímo v prohlížeči přes Binder (odznak níže; konfigurace v `.binder/`);
její data jsou uložena v balíčku OSF k souvisejícímu článku (odkaz bude
zveřejněn s publikací článku, do té doby na vyžádání) a do Binderu se
nahrávají ručně, protože nesou jména anotátorek. Notebooky Studií 1 a 2
vyžadují neveřejné řezy korpusu; k nim slouží zmrazené HTML reporty
ve `vystupy/reporty/`.

Chyby a neshody čísel hlaste v Issues tohoto repozitáře.
